# Include tag
```Django
<h1>Hello</h1>
<p>This page contains a footer in a template.</p>

{% include 'base/footer.html' %} 
```

## sending context

```Django
<div>HOME | {{ me }} | ABOUT | FORUM | {{ sponsor }}</div>
```
