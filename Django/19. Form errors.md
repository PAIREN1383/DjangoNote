# Example of form errors
```Django
<h3>Write your comment</h3>
{% if form %}
	<form action="{% url 'blog:article_detail_name' article.id%}" method="post" enctype="multipart/form-data">
		{% csrf_token %}
		{{form.as_p}}
		
		{% if form.errors %}
		<br>
			<h4>Form Errors</h4>
			{{form.errors}}
		{% endif %}
		
		<button type="submit">Submit</button>
	</form>
{% endif %}
```

# Error of one input
```Django 
<form action="" method="post" enctype="multipart/form-data">
	<label>Full name</label>
	{{form.name}}

	<label>text</label>
	{{form.text}}

	<label>image</label>
	{{form.image}}

	<label>email</label>
	{{form.email}}
	{% for error in form.email.errors %}
		<p>{{error}}</p>
	{% endfor %}

	<button type="submit">Submit</button>
</form>
```

# Change default errors
```Python
from django import forms
err_messages = {
	'required': 'New error: You must fill in this input'
}
class CreateCommentForm(forms.Form):
    name = forms.CharField(max_length=110, label='Full name', error_messages=err_messages)
    text = forms.CharField(max_length=550, widget=forms.Textarea)
    image = forms.ImageField(required=False)
    email = forms.EmailField(max_length=254)
```
