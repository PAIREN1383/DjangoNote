The exact method is case-sensitive.
## iexact method
```shell
>>> art2 = Article.objects.get(title__iexact="ArT1")
>>> art2.title
'art1'
```
It is not case-sensitive.

## contains method
```shell
>>> arts = Article.objects.filter(title__contains="art")
>>> arts
<QuerySet [<Article: art1>, <Article: art2>, <Article: new art>]>
>>> arts = Article.objects.filter(title__contains="arT")
>>> arts
<QuerySet [<Article: art1>, <Article: art2>, <Article: new art>]>
// Never use 'arts.delete()'. It directly delete objects in your database.
```
