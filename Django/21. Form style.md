# Giving a class to object of form

```Django
<form action="" method="post" enctype="multipart/form-data">
	<label class="class-label">Full name</label>
	{{form.name}}

	<label class="class-label">text</label>
	{{form.text}}

	<label class="class-label">image</label>
	{{form.image}}

	<label class="class-label">email</label>
	{{form.email}}

	<button type="submit">Submit</button>
</form>
```

``` Python
from django import forms


class CreateCommentForm(forms.Form):
    name = forms.CharField(max_length=110, label='Full name', widget=forms.TextInput(attrs={'class': 'form-control'}))
    text = forms.CharField(max_length=550, widget=forms.Textarea(attrs={'class': 'form-control'}))
    image = forms.ImageField(required=False)
    email = forms.EmailField(max_length=254)
```

# Giving label id to input
```Django
<form action="" method="post" enctype="multipart/form-data">

	<label for="{{form.name.id_for_label}}">Full name</label>
	{{form.name}}

	<label for="{{form.text.id_for_label}}">text</label>
	{{form.text}}

	<label for="{{form.image.id_for_label}}">image</label>
	{{form.image}}

	<label for="{{form.email.id_for_label}}">email</label>
	{{form.email}}

	<p>{{form.email.help_text}}</p>
	{% for error in form.email.errors %}
		<p>{{error}}</p>
	{% endfor %}

	<button type="submit">Submit</button>
</form>
```


# Giving custom label
```Django
<form action="" method="post" enctype="multipart/form-data">
	<label>{{form.name.label}}</label>
	{{form.name}}

	<label>text</label>
	{{form.text}}

	<label>image</label>
	{{form.image}}

	<label>email</label>
	{{form.email}}

	<button type="submit">Submit</button>
</form>
```

``` Python
from django import forms


class CreateCommentForm(forms.Form):
    name = forms.CharField(max_length=110, label='(New label) Full name')
    text = forms.CharField(max_length=550)
    image = forms.ImageField(required=False)
    email = forms.EmailField(max_length=254)
```
